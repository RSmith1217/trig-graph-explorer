# Trig Graph Explorer

An interactive math lab connecting a moving point on the unit circle to live
sine, cosine, and tangent graphs.

## Features

- Draggable unit-circle point
- Synchronized sine, cosine, and tangent graphs
- Exact values at special angles
- Degree and radian modes
- Animation, angle snapping, and keyboard controls
- Responsive, student-friendly layout

## Run locally

This is a dependency-free static site. Open `index.html` directly, or serve the
folder locally:

```bash
python3 -m http.server 4173
```

Then visit [http://localhost:4173](http://localhost:4173).
